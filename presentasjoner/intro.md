Overview
• Web technologies enable interactive
websites and applications to connect
millions of people and create billiondollar industries seemingly overnight.
• Web-sites use a rich mixture of rapidly
evolving technologies to provide secure,
interactive and personalized online
experience that today's users enjoy.
Web Servers and Web Browsers
• A Web page is stored on a Web server, which in turn makes it
available to the network
• To view a Web page, a client runs a software program called a Web
browser, which retrieves the page from the server and displays it
• (Graphical) Browsers are capable of displaying (rendering) not only
images, but also video, sound, animations, and a variety of
multimedia.
Description
• IT2805 - Web Technologies course
provides students with a broadly based
understanding of web technologies
and how they are most effectively
combined, along with hands-on
experience in their use.

Description
• Fundamental topics covered in this unit
include:
• the client/server paradigm, the internet
protocols used to transmit web pages,
• the markup languages (HTML, XHTML and
XML) used to describe document structure
• the styling language (CSS) used to control a
document's appearance.
• how web sites achieve interactivity through
client side scripting (JavaScript)

Assignment Grading
• Each assignment has exercises with very specific grading schema.
• For the grading of each exercise, two general criteria that your work
should meet are:
• Usefulness - does the program do what it is supposed to do? Does it produce
the specified outcomes, and handle error conditions correctly and elegantly?
• Usable user Interface (web pages), has the user interface designed so the client
can easily navigate and find the information he is searching for?
• Everything needs to be tested against the mainstream browsers.
Cheating
• You may discuss general techniques of web development with other
students
• You may give or receive help understanding assignments and debugging
work
• You may copy examples from the lecture notes and then change them to
meet the assignment requirements
• You may not copy code or allow anyone to copy your code. In my
experience, students who let other students copy their work usually don’t
know how to do it either.
• You may not copy text from any source for short answer questions on
exercises/labs and exams
