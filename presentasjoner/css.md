List of HTML5 tags and their uses.
1. Document Structure
<!DOCTYPE html> → Defines the HTML5 document type.
<html> → Root element of an HTML document.
<head> → Container for metadata (title, links, scripts).
<title> → Sets the title of the page (appears in browser tab).
<meta> → Defines metadata (charset, viewport, description, keywords).
<link> → Links external resources (e.g., CSS).
<style> → Embeds CSS styles.
<script> → Embeds or links JavaScript.
<noscript> → Content shown if JavaScript is disabled.
2. Content Sectioning
<body> → Main container for all visible content.
<header> → Introductory section (logo, navigation, title).
<nav> → Defines navigation links.
<main> → Represents main unique content.
<section> → Groups related content.
<article> → Independent content (e.g., blog post, news article).
<aside> → Side content (ads, sidebars, related info).
<footer> → Footer section (copyright, contact, links).
<address> → Contact information.
<h1>–<h6> → Headings (h1 is most important, h6 least).
3. Text Content
<p> → Paragraph.
<hr> → Horizontal rule (thematic break).
<pre> → Preformatted text (preserves spaces & line breaks).
<blockquote> → Long quotation.
<q> → Short inline quotation.
<br> → Line break.
<ol> → Ordered list.
<ul> → Unordered list.
<li> → List item.
<dl> → Description list.
<dt> → Term in a description list.
<dd> → Description in a description list.
<figure> → Container for self-contained content (image, diagram).
<figcaption> → Caption for <figure>.
<div> → Generic container.
<span> → Inline container for text.
4. Inline Text Semantics
<a> → Hyperlink.
<em> → Emphasis (italic by default).
<strong> → Strong importance (bold by default).
<small> → Smaller text.
<mark> → Highlighted text.
<abbr> → Abbreviation with tooltip.
<cite> → Reference (book, article, etc.).
<code> → Inline code.
<kbd> → Keyboard input.
<samp> → Sample output.
<var> → Variable in math or programming.
<time> → Represents a date/time.
<b> → Bold text (stylistic).
<i> → Italic text (stylistic).
<u> → Underlined text.
<s> → Strikethrough text.
<sup> → Superscript.
<sub> → Subscript.
5. Multimedia
<img> → Image.
<map> → Defines an image map.
<area> → Defines clickable areas in <map>.
<audio> → Embeds audio.
<video> → Embeds video.
<source> → Specifies media source for <audio> / <video>.
<track> → Subtitles or captions for <video>.
<canvas> → Container for graphics (via JavaScript).
<svg> → Vector graphics.
<iframe> → Inline frame (embed external page).
<embed> → Embeds external content (plugins).
<object> → Embeds external resource.
<param> → Parameters for <object>.
6. Forms & Input
<form> → Creates a form.
<input> → Input field (text, number, password, etc.).
<textarea> → Multi-line text input.
<button> → Button (clickable).
<select> → Drop-down list.
<option> → Option in a drop-down.
<optgroup> → Group of options in a drop-down.
<label> → Label for input.
<fieldset> → Groups related inputs.
<legend> → Caption for <fieldset>.
<datalist> → Predefined options for <input>
<output> → Displays output of calculations.
<progress> → Progress bar.
<meter> → Measurement within a known range.
7. Tables
<table> → Table.
<caption> → Table title.
<tr> → Table row.
<td> → Table data cell.
<th> → Table header cell.
<thead> → Table header group.
<tbody> → Table body group.
<tfoot> → Table footer group.
<col> → Column properties.
<colgroup> → Group of columns.
8. Interactive & Scripting
<details> → Expandable content.
<summary> → Summary for <details>.
<dialog> → Dialog box / popup.
<menu> → Menu list.
<menuitem> (deprecated) → Command in a menu.
<slot> → Placeholder in web components.
<template> → Template for reusable HTML.
