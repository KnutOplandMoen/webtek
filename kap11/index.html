01 · Helhetsbilde

Hvorfor XML?

Skal skrives

XML deler syntaksfilosofi med HTML, men er for data, ikke utseende. Brukes i SOAP, RSS, SVG, og masse legacy-systemer. JSON tok over mye, men XML lever videre.

Kilde: pensum 1.14.

02 · Mental modell

Som HTML, men strengere

Skal skrives

Sammenlign HTML og XML side ved side: tags, attributter, hierarki. Forskjellen er at XML krever lukke-tag, case-sensitive, kvotering, og at egne tag-navn er greit.

03 · Reisen

Et note-dokument

Skal skrives

Stegvis: <?xml version="1.0"?> → root-element → barn-elementer → attributter → CDATA-blokk → XML-validering.
