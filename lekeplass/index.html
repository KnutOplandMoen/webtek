Lekeplassen

Skriv HTML, CSS og JavaScript i hvert sitt felt og kjør koden live i en sandboxet preview. Bra for å teste CSS-selektorer, DOM-events eller en rask console.log-snutt mens du leser pensum.

Klar
HTML — struktur
CSS — styling
JS — oppførsel
Preview — sandboxet iframe

Tab setter inn to mellomrom. JavaScript-feil vises nederst i preview. Sandboxen er isolert (allow-scripts) — den kan ikke nå denne siden eller lagre noe. Endringer huskes i nettleserens localStorage.
