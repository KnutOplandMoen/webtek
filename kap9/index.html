01 · Helhetsbilde

Hvorfor JavaScript?

Skal skrives

HTML strukturerer, CSS styler, JavaScript reagerer. Uten JS er en side bare lesestoff. Vis kontrast med en knapp som ikke gjør noe — og en som faktisk endrer DOM-en.

Kilde: pensum 1.11.

02 · Mental modell

Et script kjører i sin tur

Skal skrives

JS leses ovenfra og ned, linje for linje. Variabler, funksjoner og kontrollstruktur følger samme mønster som de fleste språk — men noen ting er rare (typecoercion, hoisting, var vs let).

03 · Reisen

Fra console.log til funksjon

Skal skrives

Stegvis: skript-tag i HTML → console.log → variabler → kontrollstruktur → arrays → objekter → funksjoner → JSON. Gradvis bygging av forståelse.

05 · Begreper

Nøkkelbegreper

let / const / var
Foretrukket: const > let. Unngå var.
==
Type-koersjon, vil gi overraskelser.
===
Streng likhet — bruk denne.
undefined
Variabel deklarert, ikke tildelt.
null
Bevisst tom verdi.
truthy / falsy
0, '', null, undefined, NaN er falsy.
array vs object
Liste med rekkefølge vs nøkkel-verdi-par.
JSON
Tekstformat for å overføre objekter.

08 · Vanlige feller

Vanlige feller

Skal skrives
