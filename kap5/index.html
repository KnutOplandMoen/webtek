01 · Helhetsbilde

Hvorfor Flexbox og Grid?

Skal skrives

Før Flexbox/Grid brukte folk float, position og tabeller for layout. Det var skjørt og uforutsigbart. Vis kontrasten med en konkret «center vertical»-oppgave før og etter.

Kilde: slides presentasjoner/grid_flebox.md + presentasjoner/exam_info.md linje 1–90.

02 · Mental modell

1D versus 2D

Skal skrives

SVG-figur som viser flexbox (én rad eller én kolonne) versus grid (rader OG kolonner samtidig). Forklar main axis og cross axis for flex; rows og columns for grid.

03 · Reisen

Bygg en side-layout

Skal skrives

Stegvis: page-grid (header, sidebar, main, footer) i Grid → kort-rad i Flexbox → meny i Flexbox → galleriet i Grid. Vis at du nesten alltid bruker begge sammen.

05 · Begreper

Nøkkelbegreper

Main axis
Flex-aksen som bestemmes av flex-direction.
Cross axis
Aksen vinkelrett på main.
Grid line
De numbered linjene mellom celler.
Grid track
Mellomrom mellom to linjer (kolonne eller rad).
Grid area
Rektangulært område dekket av flere celler.
fr-enheten
Fraksjon av tilgjengelig plass i Grid.
justify-content
Plassering langs main axis (flex) / inline-axen (grid).
align-items
Plassering på cross axis.
