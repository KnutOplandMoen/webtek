Kapittel 3 · Underkapittel 1.6.10

Borders — tegn rammen

En border er den synlige rammen rundt boksen, mellom padding og margin. Tre egenskaper styrer den — bredde, stil og farge — og hver side kan settes individuelt.

01 · Pensum 1.6.10

De tre egenskapene

En ramme har alltid tre ting:

border-width
Hvor tykk rammen er. F.eks. 1px, 4px, thin, thick.
border-style
Hvordan den ser ut: solid, dashed, dotted, double, none, hidden, groove, ridge, inset, outset.
border-color
Fargen — samme syntaks som color: navn, hex, eller rgb/rgba.
Viktig

Standard border-style er none. Hvis du bare setter border-width og border-color uten en stil, vises ingenting. Det er en klassisk feilkilde.

MCQ · 1.6.10
Hvilken egenskap MÅ du oppgi for at en ramme i det hele tatt skal vises?
  • border-width
  • border-style
  • border-color
  • border-radius
Standard er border-style: none — uten en eksplisitt stil, vises ingenting.
02 · Shorthand

Shorthand-skrivemåten

Du kan kombinere alle tre egenskapene på én linje med border-shorthanden. Rekkefølgen er: width style color. 

.kort {
  border: 4px dashed #0060A3;
}

/* tilsvarer: */
.kort {
  border-width: 4px;
  border-style: dashed;
  border-color: #0060A3;
}
MCQ · 1.6.10
Hvilken shorthand gir en 2 piksler tjukk, prikket, rød ramme?
  • border: red 2px dotted;
  • border: 2px dotted red;
  • border-style: 2px dotted red;
  • border: dotted 2px red;
Pensums rekkefølge: width, style, color. Praktisk talt er rekkefølgen fleksibel i moderne nettlesere, men eksamen vil at du kan «riktig» rekkefølge.
03 · Per side

Forskjellig stil per side

Du kan adressere bare én side ved å sette inn -top, -right, -bottom eller -left i egenskapsnavnet: 

.kort {
  border-bottom: 2px solid #c9c0ae;   /* bare bunnstrek */
  border-top-style: dashed;
  border-left-color: red;
}

Eller bruke shorthand med fire verdier — på samme måte som padding og margin (klokkeretningen fra topp): 

.kort {
  border-style: solid dashed dotted double;
  /* topp solid, høyre dashed, bunn dotted, venstre double */
}
MCQ · 1.6.10
Hva tegner border-bottom: 2px solid black;?
  • Ramme på alle fire sider
  • Bare en svart strek under elementet
  • En understreking av teksten
  • En skygge nederst
Suffikset -bottom begrenser til den siden. Praktisk for skille-streker uten å boks-omslutte hele elementet.
04 · Avrunding

Avrundete hjørner

Egenskapen border-radius avrunder hjørnene på boksen. Verdien kan være piksler, prosent eller andre lengder. Pensum nevner den som måten å lage runde knapper, kort og avatarer på. 

.knapp { border-radius: 4px; }
.avatar { border-radius: 50%; }   /* en sirkel, hvis bredde = høyde */
Tips

border-radius: 50% på et kvadrat = perfekt sirkel. Det er hele trikset bak runde profilbilder på de fleste nettsteder.

MCQ · 1.6.10
Hva gjør border-radius: 50%; på en kvadratisk boks?
  • Halverer bredden
  • Gjør boksen til en sirkel
  • Roterer rammen 50 grader
  • Skjuler halvparten av rammen
50 % radius på alle hjørner av et kvadrat = perfekt sirkel. Med ulik bredde og høyde får du en ellipse.
05 · Test deg selv

Avsluttende spørsmål

MCQ · Helhet
Hvor mange ulike border-style-verdier nevner pensum?
  • Fire (none, solid, dashed, dotted)
  • Ti eller fler — bl.a. none, solid, dashed, dotted, double, groove, ridge, inset, outset
  • Bare to (solid og dotted)
  • Det finnes ingen border-stil
Pensum lister flere — de fleste brukes sjelden i praksis, men du må kjenne til dem på eksamen.
MCQ · Helhet
Hva er standardverdien for border-style?
  • none
  • solid
  • dotted
  • inherit
Derfor må du eksplisitt sette en stil for at rammen skal vises. Bare bredde og farge er ikke nok.
MCQ · Helhet
Hvilken kort skrivemåte gir samme resultat som border-width: 1px; border-style: solid; border-color: black;?
  • border: black solid 1px;
  • border: 1px solid black;
  • border: solid;
  • border-all: 1px black;
Pensum-rekkefølgen er bredde → stil → farge.
