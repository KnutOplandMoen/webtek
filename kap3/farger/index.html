01 · Pensum 1.6.7

Hvor i CSS brukes farger?

De to vanligste egenskapene er color (tekstfarge) og background-color (bakgrunnsfarge på elementets boks). Begge tar én fargeverdi, og den verdien kan oppgis på tre måter ifølge pensum. 

p {
  color: blue;             /* navn   */
  background-color: #f4f1ea; /* hex   */
}
.varsel {
  background-color: rgba(198, 138, 31, 0.2); /* rgba   */
}
02 · Fargenavn

147 navngitte farger

Pensum nevner at nettleseren forstår 147 forhåndsdefinerte fargenavn, f.eks. red, blue, cyan, orange, lightgoldenrodyellow. Greit å bruke i eksempler eller når du raskt vil teste, men i designsammenheng vil du som regel ha mer presis kontroll.

Når brukes navn?

I undervisning og prototyper. I produksjonskode brukes hex eller rgba — fordi de gir nøyaktig kontroll over fargen og kan kopieres direkte fra designverktøy som Figma eller Adobe.

MCQ · 1.6.7
Cirka hvor mange forhåndsdefinerte fargenavn nevner pensum at nettleseren støtter?
  • 16
  • 64
  • 147
  • Over 1000
147 navn — fra aliceblue til yellowgreen. De originale 16 er en delmengde.
03 · Hex

Hex-koder

En hex-kode er # etterfulgt av seks heksadesimale sifre (0–9, A–F). Den koder for tre tall: rødt, grønt og blått, hvert fra 00 (av) til FF (255, max). 

#000000   /* svart — alle null  */
#FFFFFF   /* hvit  — alle max   */
#FF0000   /* rent rødt          */
#0060A3   /* en bestemt blå     */

Det finnes også en kortform med tre sifre, der hvert siffer dobles: #0AC betyr #00AACC. Pensum nevner at begge formene er gyldige.

MCQ · 1.6.7
Hvilken farge representerer #000000?
  • Hvit
  • Rødt
  • Svart
  • Gjennomsiktig
Alle tre kanaler (rød, grønn, blå) på 0 = ingen lys = svart. Motsatt: #FFFFFF = hvit.
MCQ · 1.6.7
Hva representerer den midterste delen i hex-koden #1F4E79?
  • Den røde komponenten
  • Den grønne komponenten
  • Den blå komponenten
  • Alfa-kanalen
Rekkefølgen er rød (1F), grønn (4E), blå (79) — RR GG BB.
04 · RGB / RGBA

RGB og RGBA

rgb() tar tre tall, hver mellom 0 og 255, for rødt, grønt og blått. rgba() tar et fjerde tall: alfa, mellom 0 (helt gjennomsiktig) og 1 (helt ugjennomsiktig). 

rgb(255, 0, 0)            /* rent rødt           */
rgba(0, 96, 163, 0.5)     /* halvgjennomsiktig blå */
rgba(0, 0, 0, 0)          /* usynlig svart        */

Alfa-kanalen er den eneste måten i pensum å lage en gjennomsiktig farge på (utenom opacity-egenskapen, som påvirker hele elementet). Brukes typisk for skygger, overlegg og glass-aktige bakgrunner.

MCQ · 1.6.7
Hva betyr den fjerde verdien i rgba(0, 96, 163, 0.5)?
  • Lysstyrken
  • Opasiteten (alfa) — 0.5 = 50 % gjennomsiktig
  • Mengden hvit som blandes inn
  • En ID for fargen
Alfa går fra 0 (usynlig) til 1 (full opasitet). Pensum bruker det aktivt for halv-gjennomsiktige bakgrunner.
MCQ · 1.6.7
Hvilken verdi-rekkevidde har en enkelt rgb-kanal?
  • 0 til 1
  • 0 til 255
  • 0 til 100
  • −128 til 127
256 nivåer per kanal. Det matcher to heksadesimale sifre — derfor passer det perfekt med hex-formatet.
05 · Kontrast

Kontrast mellom tekst og bakgrunn

Pensum 1.6.7 påpeker eksplisitt at lav kontrast mellom tekst og bakgrunn gjør siden vanskelig å lese — særlig for synshemmede. Lys grå tekst på hvit bakgrunn ser pent ut, men er ofte uleselig på solfylt mobilskjerm.

Tommelfingerregel

Sikt på minst kontrastforhold 4.5:1 mellom tekst og bakgrunn (WCAG AA). Verktøy som Adobe Color, WebAIM Contrast Checker eller nettleserens DevTools kan måle dette automatisk.

MCQ · Helhet
Hvilke tre måter å oppgi farger på lister pensum?
  • Hex, navn, hsl
  • Hex, navn, rgb/rgba
  • CMYK, RGB, navn
  • Pantone, hex, hsla
Pensum 1.6.7 lister disse tre. hsl() og andre fargemodeller finnes i nyere CSS, men er ikke i pensum.
MCQ · Helhet
Hvilken hex-kode tilsvarer rgb(255, 255, 255)?
  • #000000
  • #888888
  • #FFFFFF
  • #FF00FF
FF hex = 255 dec. Tre kanaler på 255 = full hvitt.
