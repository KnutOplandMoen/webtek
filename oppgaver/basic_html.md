Basic HTML Tasks
Task 1: Create a basic HTML page structure (`<!DOCTYPE html>`, `<html>`, `<head>`,
`<body>`).
Task 2: Add a title and your first heading `<h1>`.
Task 3: Write a few paragraphs using `<p>`, `<br>`, and `<hr>` for breaks.
Task 4: Use all heading levels (`<h1>` to `<h6>`).
Task 5: Format text with `<b>`, `<i>`, `<u>`, and `<mark>`.
Task 6: Add a famous quote with `<blockquote>` and `<q>`.
Task 7: Create an unordered list of favorite foods.
Task 8: Create an ordered list of your daily routine.
Task 9: Make a nested shopping list (e.g., Fruits → Apples, Bananas).
Task 10: Use HTML entities to display symbols (`©`, `♥`, `→`).
Task 11: Add links to 3 of your favorite websites (one should open in a new tab).
Task 12: Create a long page with multiple sections and a “Back to Top” link.
Task 13: Make a navigation list with links that scroll to different sections on the same page.
Task 14: Insert an image with proper `alt` text.
Task 15: Adjust image size with `width` and `height`.
Task 16: Add a caption to an image using `<figure>` and `<figcaption>`.
Task 17: Build a simple table showing your weekly schedule.
Mini Project
Build a single-page "travel blog" with these elements:
Header
A main title (e.g., "My Travel Adventures").
A short tagline under it.
Navigation Bar
Links to sections like Home, Destinations, Gallery, Contact.
Main Content
An <article> about your favorite travel destination with headings and paragraphs.
A <blockquote> with a travel-related quote.
At least one image with a caption (<figure> + <figcaption>).
List Section
An unordered list of places you want to visit.
An ordered list of your top 3 favorite trips so far.
Table
A small table showing your past trips with columns: Destination, Year, Highlight.
Footer
Add a copyright notice (© 2025 Your Name).
Add a "Back to Top" link.
