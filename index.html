Følg dem i rekkefølge eller hopp dit du trenger oppfriskning. Hvert kapittel har samme oppbygning: helhetsbilde, dypdykk, vanlige feller, quiz og kapittel-eksamen i flashcard-form.

KAP 01
Web og Internett
Klient/tjener, HTTP, DNS, TCP/IP og hva en nettleser faktisk gjør.
 KAP 02
HTML — struktur og semantikk
Elementer, attributter, tekst, lister, lenker, bilder, tabeller, semantiske tagger.
 KAP 03
CSS — grunnleggende
Selektorer, kaskade, arv, farger, fonter, margin, padding, pseudo-klasser.
 KAP 04
CSS — videregående
Display, document flow, overflow, mer pseudo, og hvordan boksene egentlig oppfører seg.
 KAP 05
Layout — Flexbox og Grid
1D-flexbox og 2D-grid: når brukes hva, og hvordan beskrives layout deklarativt.
 KAP 06
Responsivt webdesign
Media queries, breakpoints, mobile first og fluide enheter.
 KAP 07
Multimedia på web
Bilder, video, audio: formater, attributter og tilgjengelighet.
 KAP 08
Forms og validering
Form-elementet, input-typer, klientside-validering og god skjema-design.
 KAP 09
JavaScript — grunnleggende
Datatyper, variabler, kontrollstruktur, arrays, objekter, funksjoner og JSON.
 KAP 10
JavaScript — videregående
Scope, tid, DOM-manipulering og events i et dynamisk dokument.

Pensum og kilder

Om emnet

IT2805 Webteknologi er et grunnleggende emne ved NTNU som dekker hvordan nettsider er bygd opp av HTML, CSS og JavaScript, og hvordan de henger sammen via web-arkitekturen og protokollene som binder Internett sammen.

Pensum er læreboka og forelesningene. Hovedkilden er it2805.gitbook.io/curriculum, supplert av slides og øvinger fra kursets BlackBoard-side.

Pensum

  • Forelesningsoversikt med slides og kapittel-mapping.
  • Lærebok: it2805.gitbook.io/curriculum (gratis, gitbook).
  • Slides per emne ligger i presentasjoner/ i prosjektroten.
  • Øvinger ligger i oppgaver/ som markdown-utdrag.
  • Tidligere eksamener i tidligere_eksamener/.
Bidra

Siden lever av tilbakemelding. Finner du noe som er feil, uklart eller mangler? Si fra — det er alltid noen som lurer på det samme. Kapitlene fylles inn over tid; tomme seksjoner er der med vilje for å vise hva som mangler å skrives.
